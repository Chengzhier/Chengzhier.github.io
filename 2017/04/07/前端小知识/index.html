绝对路径和相对路径
  • 绝对地址：相对于磁盘的位置去定位文件的地址

CSS盒子模型

把元素叫做盒子，设置对应的样式分别为：盒子的宽度(width)、盒子的高度(height)、盒子的边框(border)、盒子内的内容和边框之间的间距(padding)、盒子与盒子之间的间距(margin).

设置宽高
width:200px;设置盒子的宽度，次宽度是指盒子内容的宽度，不是盒子整体的宽度
height:200px;设置盒子的高度，次高度是指盒子内容的高度，不是盒子整体的高度

设置边框
设置一边的边框如顶部边框

border-top-style:solid;设置顶部边框的线性为实线，常用的有：solid(实线)dashed(虚线)dotted(点线)
可以简写为border-top:10px solid red;
设置左边框
border-left-color:red;设置左边框颜色为红色
border-left-width:10px;设置左部边框粗细为10px
border-left-style:solid;设置左部边框的线性为实线，常用的有：solid(实线)dashed(虚线)dotted(点线)
可以简写：border-left：10px solid red;
设置右边框

border-right-width:10px;设置右部边框粗细为10px
border-right-style:solid;设置顶右边框的线性为实线，常用的有：solid(实线)dashed(虚线)dotted(点线)
可以简写：border-right：10px solid red;

如果四个边设置一样，可以将四个边的设置合并成一句话

css元素 溢出：

块元素、内联元素、内联块元素
元素就是标签，布局中常用的三种标签，块元素、内联元素、内联块元素
1、块元素
块元素，也可以称为行元素，布局常用的标签有div、p、ul 、li、h1-h6、dl、dt、dd等等它们的布局
特点：

  • 支持全部的样式
  • 如果没有设置宽度，默认的宽度为父级宽度100%
  • 盒子占据一行、即使设置了宽度

  • 2、内联元素
    内联元素，也可以称为行内元素，布局中常用的标签有a、span、em、b、strong、i
    特点：

  • 支持部分样式(不支持宽、高、margin上下、padding上下)

  • 宽高由内容决定
  • 盒子并在一行
  • 代码换行，盒子之间会产生间距
  • 子元素是内联元素，父元素可以用text-align属性设置子元素内容的水平对其方式

解决内联元素间隙的方法
1、去掉内联元素之间的换行
2、将内联元素的父级设置font-size为0，内联元素自身在设置font-size

3、内联块元素
内联块元素，也交行内块元素，是新增的元素类型，现有元素没有归于次类别的，img和input元素的行为类似这种元素，但是也归类于内联元素，我们可以用display属性将块元素或者内联元素转化为这种元素。
特点：

  • 支持全部样式
  • 如果没有设置宽高，宽高由内容决定
  • 盒子并在一行
  • 代码换行，盒子会产生间距
  • 子元素是内联块元素，父元素可以用text-align属性设置子元素水平对其方式。
    这三种元素，可以通过display属性来相互转化，块元素用的比较多，所以我们经常把内联元素转化为块元素，少量转化为内联块，而要使用内联元素时，直接使用内联元素，而不用块元素转化。
    display属性
    display属性是用来设置元素的类型及隐藏的常用属性有：
    1、none 元素隐藏且不占位置
    2、block 元素以块元素显示
    3、inline 元素以内联元素显示
    4、inline-block 元素以内联块元素显示

浮动
浮动特性
1、浮动元素有左浮动(float:left)和右浮动(float:right)两种
2、浮动的元素会向左或向右浮动，碰到父元素边界、其他元素才停下来
3、相邻浮动的块元素可以并在一行，超出父级宽度就换行
4、浮动让行内元素或块元素自动转化为行内快元素(此时不会有行内块元素间隙问题)
5、浮动元素后面没有浮动的元素会占据浮动元素的位置，没有浮动的元素内的文字会避开浮动的元素，形成文字绕图的效果
6、父元素如果没有设置尺寸（一般是高度不设置），父元素内整体浮动的元素无法撑开父元素，父元素需要清除浮动
7、浮动元素之间没有垂直margin的合并

清除浮动：

  • 父级上增加属性overflow：hidden(元素溢出)
  • 在最后一个子元素的后面加上一个空div，给它样式属性clear：both(不推荐)
  • 使用成熟的清浮动样式类，clearfix
  • .clearfix:attter,.clearfix:before{
  • content:””,display:table;}
  • .clearfix:afther{clear:both;}
  • .clearfix{zoom:1;}

清除浮动的使用方法：
.con2{… overflow:hidden}
< div class=”con2 clearfix”>

list-style: none;去掉无序列表中的点

文档流
文档流，是指盒子按照html标签写的顺序依次从上到下，从左到右排列，块元素占一行，行内元素在一行之内从左到右排列，先写的先排列，后写的排在后面，每个盒子都占据自己的位置。
关于定位
我们可以使用css的position属性来设置元素的定位类型，postion的设置项如下：

  • relative 生成相对定位元素，元素所占据的文档流的位置保留，元素本身相对自身原位置进行偏移
  • absolute 生成绝对定位元素，元素脱离文档流，不占据文档流的位置，可以理解为漂浮在文档流的上方，相对于上一个设置了定位的父级元素来进行定位，如果找不到，则想对于body元素进行定位。
  • fixed生成固定定位元素，元素脱离文档流，不占据文档流的位置，可以理解为漂浮在文档流的上方，相对于浏览器窗口进行定位。
  • static 默认值，没有定位，元素出现在正常的文档流中，相当于取消定位属性或者不设置定位属性。

  • inherit 从父元素继承position 属性的值。

    定位元素的偏移
    定位的元素还需要用left、right、top或者bottom来设置相对于参照元素的偏移值。

    定位元素层级
    定位元素是浮动的正常的文档流之上的，可以用z-index属性来设置元素的层级

定位元素特性
绝对定位和固定定位的块元素和行内元素会自动转化为行内块元素

CSS3 权重的等级

CSS权重指的是样式的优先级，有两条或多条样式作用于一个元素，权重高的那样样式对元素起作用，权重相同，后写的样式会覆盖前面写的样式。

权重的等级

1、!important,加在样式属性值后，权重值1000
2、内联样式，如：style=””,权重值为1000
3、ID选择器，如#content,权重值为100
4、类，伪类和属性选择器，如：content、:hover权重值为10
5、标签选择器和伪元素选择器，如：div、p\：before权重为1
6、通过选择器( x）、子选择器(>)、相怜选择器(+)、同胞选择器(~)、权重值为0
