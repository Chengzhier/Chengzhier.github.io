传统的正则表达式

  • Regular Expression的“Regular”一般被译为“正则”、“正规”、“常规”。此处的“Regular”即是“规则”、“规律”的意思，Regular Expression即“描述某种规则的表达式”之意

概要

正则表达式在python中简单的使用，正则表达式又称正规表达式、正规表示法、正规标示式、常规表示法（英语：Regular Expression，在代码中常简写为regex、regexp或者RE）是计算机科学的一个概念。正则表达式使用单个字符串来描述、匹配一系列匹配某个句法规则的字符串。在很多文本编辑器里，正则表达式通常用来检索、替换那些匹配某个模式的文本。

模块

在python中需要通过正则表达式对字符串匹配的时候，可以使用一个模块为re

  • re模块需要先导入（import re）
    • 使用方式为：result = re.match(正则表达式，要匹配的字符串)
    • re.match是用来进行匹配检查的方法，若字符串匹配正则表达式，则match方法返回匹配对象（Match Object）,否则返回None(注意不是空字符串)
    • 匹配对象Match Object 具有group方法，用来返回字符串的匹配部分。
    • re.match()能够匹配以xx开头的字符串
    • match方法默认匹配都是从左到右，找到一个就返回，不会保存多个值

\S 匹配非空白
\w 匹配单词字符，即a-z、A-Z、0-9、下划线
\W 匹配非单词字符

2.匹配[]中列举的字符

注意：因为match方法是从左往右查找所以如果第一个字符如果不符合要求就会直接报错
3.匹配数字\d

4.匹配非数字\D不是数字的

5.匹配空白字符串\s(包含tab键）

6.匹配非空白\S

7.匹配单词字符\w，即a-z、A-Z、0-9、下划线

8.匹配非单词字符\W

表示数量

介绍：可以匹配多个字符的相关格式

字符 功能
* 匹配前一个字符出现0次或者是无限次，即可表示可有可无
+ 匹配前一个字符出现1次或者是无限次，即至少1次
？ 匹配前一个字符出现1次或者是0次，即要么有1次要么没有
{m} 匹配前一个字符出现m次
{m,} 匹配前一个字符至少出现m次
{m,n} 匹配前一个字符出现m到n次

1.匹配前一个自读出现0次或者是无限次，即表示可有可无
需求：匹配出一个字符串第一个字母为大小字符，后面都是小写字母并且这些小写字母可有可无

2.匹配前一个字符出现1次或者是无限次，至少出现1次+
需求：匹配出，变量名是否有效

3.匹配前一个字符出现1次或者是0次，要么有1次要么没有
需求：匹配出0-99之间的数字

4.匹配前一个字符出现m次{m}
需求：匹配出，8-20位的密码，可以是大小写英文字母、数字、下划线

5.匹配前一个字符最少出现m次{m,}

6.匹配前一个字符出现从M到N次{m,n}

search 可以直接搜索数字

findall 统计所有的数字并且返回的是一个列表

需求：统计pyhhon 、c、c++相应文章的阅读次数

需求：将匹配的数字更换

split 根据匹配进行切割字符串，并返回一个列表

总结：

python里数量词默认的是贪婪的（在少数语言里也可能是默认非贪婪），总是尝试匹配尽可能多的字符
非贪婪则是相反，总是尝试匹配尽可能少的字符

正则表达式，模式中使用通配字，从左到右的顺序求值时，会尽量“抓取”满足匹配最长字符串。
解决方式：非贪婪操作符“？”
